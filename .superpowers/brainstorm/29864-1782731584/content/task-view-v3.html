v2 → v3 关键变化（对齐真 Codex 截图）

Master 左下：加 Settings（Codex 底部）

  • 当前用户（avatar 字母 chip + 用户名 + role/workspace）
  • 右侧 ⚙ icon → 设置入口（profile / theme / logout）
  • border-top 1px 视觉断开 main content

Detail 右上：WORKSPACE（Codex 环境信息）

  • 比照 Codex 的"env vars"——但语义不是 OS env，而是 workspace context：goal / board / user / runtime / api / tokens
  • mono 字体 + 64px label + flex 1 value（text-overflow ellipsis）
  • 底部 token bar（4px 高度 + 30% 填充）—— progress indicator

Detail 右中：AI ROLES（Codex 子智能体）

  • 从 v2 的"左侧 AI ROLES"搬到右侧——对齐 Codex 子智能体位置
  • 6 个 agent（Backlog Refiner / Todo Orchestrator / Dev Crafter / Review Guard / Done Reporter / Blocked Resolver）
  • 每行：6px state dot（mono 颜色编码）+ name + 状态文字（idle / running N s / queued）
  • Dev Crafter 当前在跑（pulse + 5.2s 计时）

Detail 右下：CARD RUNTIME + accordion

  • 保留 v1 的 Modified files / Plan / Audit accordion（task-specific runtime）
  • 前面加 CARD RUNTIME header 显示 CARD id / state / assignee —— 跟 AI ROLES 区视觉断开
  • Default 展开：Modified files（最高频需要的信息）
