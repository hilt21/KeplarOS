v1 → v2 关键变化

Master pane 改为 goal-space 分组

  • 每个 goal space 是 section header：icon（首字母）+ 名字（hover-expand overflow）+ task count（mono）
  • Tasks 缩进 16px 在 group 下面，按 status dot + display_id + title + 相对时间显示
  • 支持折叠（▾/▸）—— collapsed group 不展开 children，但仍可点 group header 跳 Goal Space 视图
  • 点 goal space section header → 切到 Goal Space 视图（kanban）
  • 点 task → 切到 Task 视图（in-place shell 切换）

Top bar 面包屑可点击

  • "Railway Metro 2026 Q1" 段可点击 → Goal Space 视图
  • "Main board" 段可点击 → Goal Space 视图（board 切换）
  • "Signaling timing" 段不可点击（当前 task，永远显示）
  • hover 高亮（background + rounded）—— 视觉提示可点击

对齐 Codex 真界面

  • Codex = conversation 列表，time-grouped（"已安排 / 今天"）
  • KEPLAR = goal-space-grouped 任务列表——把"项目"维度加进去
  • 其他保留 v1：timeline-based conversational UI、progressive disclosure、state machine breadcrumb、input 区

取舍

  • Master pane 缩进层级深（goal space 16px + task 28px），窄屏更拥挤——可能 240px 改 280px
  • Goal space 数量多时（10+）master 滚动量大——后续可加 virtualization
